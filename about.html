About Tinkle

Tinkle is a modern, lightweight social media platform prototype built for demonstration purposes. It simulates core social features such as user profiles, feeds, likes, comments, and following, all on the frontend.

  • Browse a feed of posts from all users
  • View and interact with user profiles
  • Like, comment, and follow/unfollow users
  • All data is simulated for demo purposes

Tinkle is inspired by the clean, modern design of platforms like Instagram and Twitter, and is fully responsive for mobile and desktop use.
